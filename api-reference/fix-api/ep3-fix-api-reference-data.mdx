Participants may download a list of the instruments available (or reference data about them) to trade on the EP3 Platform by submitting a SecurityListRequest [x] message.

Upon receipt, the EP3 Platform will respond with a SecurityList [y] message containing the requested information. Note that the content provided by the SecurityList [y] message does not differ depending on whether an individual symbol was listed versus all securities.
