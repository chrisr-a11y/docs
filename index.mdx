This guide provides a comprehensive overview of the Polymarket Exchange's platform functionality, market structure, and API architecture. Please note that this documentation is specific to the Polymarket Exchange.

## Quick Health Check

Verify connectivity to the API with a simple health check:

**REST API:**
```bash
curl https://api.prod.polymarketexchange.com/v1/health
```

**gRPC:**
```bash
grpcurl grpc.prod.polymarketexchange.com:443 polymarket.v1.HealthAPI/HealthCheck
```

**Response:**
```json
{"status": "ok"}
```
